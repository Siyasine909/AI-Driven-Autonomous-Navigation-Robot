# AI-Driven Autonomous Navigation Robot

## Description
Autonomous mobile robot integrating computer vision (OpenCV) and LiDAR-based obstacle avoidance for real-time navigation.

## Features
- Vision-based object tracking
- LiDAR obstacle detection
- Embedded multi-motor control
- Serial communication (Python ↔ Arduino)

## Technologies
- Python (OpenCV)
- Arduino (C/C++)
- LiDAR
- Embedded Systems
